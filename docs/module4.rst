:math:`a * b = c`

* a - это первое ваше число. Если укажите str значение, скрипт запросит ввести заново a.
* b - это второе ваше число. Если укажите str значение, скрипт запросит ввести заново b.

**Пример:**

a = 2

b = 2

c = 4
